# Individual Project Analysis - Archit Jaiswal

## 1: New e-Commerce site

> **Selected SDLC**: Incremental Development

**Justification**: The company wants to add cool features like custom colors and design tools. Starting with basic shopping first and then adding features step by step makes sense. This lets them test with real customers early and adjust based on feedback. It also helps manage the mixed team of experienced developers and new interns.
